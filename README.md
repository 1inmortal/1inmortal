## 🤝 Cómo colaborar
1. Haz fork y crea una rama: `feat/lo-que-sea`.
2. Usa **Conventional Commits** (`feat:`, `fix:`, `chore:`…).
3. Asegura linters/tests: `pnpm test` / `npm run test`.
4. PR descriptivo con screenshots/GIFs.
